Something about configuration

VPS：

  • server refers to the public ip address, e.g. 138.128.206.46

  • server port refers to the serverport of the client, in this post, the port is 8888
    Client：

  • serverIP correspond to the server of the VPS;
  • port correspond to the server port of the VPS

Something about damn CentOS

  1. the yum of centos is based on python2.7, however pip is based on python 3.6 So you may need:

    sudo yum install centos-release-scl

    sudo yum install rh-python36 scl enable rh-python36 bash

    to install shadowsocks using pip3.
