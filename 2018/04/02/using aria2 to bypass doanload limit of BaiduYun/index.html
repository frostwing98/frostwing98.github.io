using aria2 to bypass doanload limit of BaiduYun

2018-04-02
  • Daily

<!doctype html> Problem: there is no Baiduyun client on linux… 管他呢，垃圾百度云限我速度，早知道就不送这么多钱了……
1.chrome://extensions，安装baiduexporter与aria2c 之后打开百度云可以看到多了一个导出下载选项。

  1. apt-get install aria2 配置 最小限制：10M json-rpc链接：http://127.0.0.1:6800/jsonrpc
  2. aria2.sh中写入：

aria2c –enable-rpc –rpc-listen-all –rpc-allow-origin-all –file-allocation=none –max-connection-per-server=3 –max-concurrent-downloads=5 –continue -d ~/ 这个脚本在后台运行aria2.

选中文件，导出下载，就可以看到速度飞升。 4.http://binux.github.io/yaaw/demo/ 扳手那里设置127.0.0.1:6800就可以在线看到。
