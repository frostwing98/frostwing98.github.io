My linux configurations

I had experienced several severe problems requiring re-installations of my system. After the mess, I decide to post and share a backup of my full configurations of my system.

1. System information

  • Linux Mint 19.3 Cinnamon
  • Partition:
    • 200m Primary partition for /boot
    • 200G Logical partition, versatile
      • 50G for /root
      • 150G for /home
    • 300G Independent Data Section mounted as /Data

2. Themes and appearances

  • Theme: Numix Daily
  • Icon: Numix Circle
  • Desktop: Tyr-himin
  • Font: ProFont for Powerline, all set to 18, window title 20
  • Panel:
    • Heiht 50
    • icon 0.75
  • Desklets:
    • CPU load
    • Calender Desklet
      • Font Size 16
      • Transparency 75

3. Applications(Terminal)

  • zsh with agnoster
  • terminal
    • Initial terminal size 80x24
    • Set font color to green
    • Transparent background to around 20 to 25
    • Palette tango
  • shadowsocks
    • install from python, not apt.
  • proxychains
  • npm
  • cpan
  • adb
  • textlive
